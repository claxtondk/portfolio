HTML and Javascript Quizzes

In order to begin this project, I had to learn the basics. I had, of course, heard of HTML and Javascript before, but had no experience working with either one. I really struggled with these tasks, but the fact that I could choose fun and personally relevant topics made the finish line seem all that closer. Using two different templates, I was able to create a standard quiz as well as a drag-n-drop assessment. I changed colors, changed sizes, and added animations that I thought suited the subjct matter. When it finally clicked for me, I was so excited!

Click on the images to access your desired quiz. Enjoy, and good luck!
